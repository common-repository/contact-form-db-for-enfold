Save All Contact from Enfold Module Contact in DB

== Description == 

Save the information compiled by the users of your website on Enfold Contact Forms. You will easily find it in a simple info panel.

== Screenshots == 

1. Contact Form Example
2. Enfold Contact Form Panel in Settings Panel
3. Enfold Contact Form Panel Example

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/contact-form-db-for-enfold` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate Contact Form DB for Enfold through the 'Plugins' screen in WordPress
3. Use the Enfold Contact Form panel in the Settings Panel to display the contact form infos.
